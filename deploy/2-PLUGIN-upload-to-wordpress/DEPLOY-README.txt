=============================================
  TABAIX AMAZON PRO - WordPress Plugin
=============================================

WHERE TO UPLOAD:
  WordPress Dashboard → Plugins → Add New → Upload Plugin

HOW TO UPLOAD:
  1. Zip this entire folder into "tabaix-amazon-pro.zip"
  2. Go to your WordPress admin dashboard
  3. Navigate to Plugins → Add New
  4. Click "Upload Plugin"
  5. Select the zip file
  6. Click "Install Now"
  7. Click "Activate"

NOTES:
  - This is a WordPress PLUGIN, not a theme
  - Install this AFTER activating the theme (folder 1)
  - This plugin adds the Amazon product scraper functionality
