=============================================
  TABAIX THEME - WordPress Theme
=============================================

WHERE TO UPLOAD:
  WordPress Dashboard → Appearance → Themes → Add New → Upload Theme

HOW TO UPLOAD:
  1. Zip this entire folder into "tabaix-theme.zip"
  2. Go to your WordPress admin dashboard
  3. Navigate to Appearance → Themes → Add New
  4. Click "Upload Theme"
  5. Select the zip file
  6. Click "Install Now"
  7. Click "Activate"

NOTES:
  - This folder contains ONLY the WordPress theme files
  - The Amazon plugin is in folder 2 (separate upload)
  - The SEO backend is in folder 3 (deploy to Vercel)
