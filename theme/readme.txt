TABAIX WordPress Theme
=======================

Professional WordPress theme for TABAIX - Free browser tools, Saudi Arabia guides, GCC deals, and performance-first architecture.

Installation
------------
1. Copy the `tabaix-theme` folder into `/wp-content/themes/`.
2. Activate the theme from WordPress admin under Appearance > Themes.
3. Configure menus, widgets, and homepage settings via Appearance.

Requirements
------------
- WordPress 6.5+
- PHP 8.2+
- MySQL 8.0+

Build
-----
npm install
npm run build

Files
-----
- `style.css` = Theme metadata and design system variables
- `functions.php` = Theme bootstrap and includes
- `front-page.php` = Homepage template
- `single-tool.php` = Tool interface template
- `inc/` = Theme helper modules
- `tools/` = Browser-based tool templates and assets

Support
-------
For setup or customization, open the theme folder in your editor and adjust the templates or CSS as needed.
